# Nonogram Solver

### Summary
This is a feature-rich nonogram solver. In addition to solving nonograms, it can also give you a hint where you can 
make the next deduction.

You can save your current state and load it at a later point in time. Your save files are stored locally in your
browser.

### How to run

1. Clone this repository OR download the ZIP and extract it into a folder
2. Run `npx serve` (or any other HTTP-Server) inside the folder containing `index.html`
3. Open `127.0.0.1:3000` (port may differ depending on your server) and enjoy!
